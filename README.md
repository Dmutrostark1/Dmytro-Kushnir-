# Cubic Spline Interpolation for Route Elevation ⛰️

Цей проєкт — лабораторна робота з моделювання рельєфу гірського маршруту (станція Заросляк – гора Говерла) за допомогою кубічних сплайнів.

## Опис проєкту
Програма автоматично завантажує координати та висоти точок маршруту з відкритого API . Після отримання даних розраховується кумулятивна відстань за допомогою формули гаверсина. Для апроксимації рельєфу між вузловими точками розроблено алгоритм побудови кубічних сплайнів з використанням методу прогонки  для розв'язку системи лінійних алгебраїчних рівнянь.

## Основний функціонал:
-  Отримання висотних даних через REST API.
-  Розрахунок відстаней між координатами .
-  Реалізація тридіагональної матриці та алгоритму прогонки з нуля.
-  Візуалізація результатів апроксимації (для 10, 15 та 20 вузлів) через `matplotlib`.
-  Аналіз маршруту: загальний набір висоти, крутизна, механічна енергія.
